ES6学习笔记

前言

学到哪里写到哪里，方便回头的时候复盘学习之路

let关键字

  • 变量不能重复声明。

    let a = 1; let a =2;

    会报错，但是var关键词不会。

  • let的作用域很小，只在代码块内实现，并且不存储在window对象里。

  • 不存在变量提升。

  • 不影响作用域链。

const声明常量

  • 值不能被修改的量被称为常量。
  • 注意事项：
    • 必须要赋一个初始值，否则报错。
    • 一般大写。
    • 对数组和对象内的修改不算对常量的修改，不会报错。（因为常量的指向地址没有改变。）

模板字符串

  • es6标准支持了新的反引号，并且支持换行符

<script> let list = `<ul> <li>炼狱小镇</li> <li>核子危机</li> <li>荒漠迷城</li> </ul>`; </script>

变量拼接

let name = 'zjl';

let out = `我的名字叫${name}`；

console.log(out)

正常输出。

箭头函数

let fn = () => {

Something...

}

特性

  • 箭头函数的this始终指向声明时所在的作用域
  • 箭头函数不能作为构造函数
  • 没有arguments变量

箭头函数的省略形式

  • 当有且只有一个形参时，可以省略小括号。

let add = n =>{

return n+n;

}

此处省略了小括号。

  • 当只有一条语句时，还可以省略花括号。

    ​ let pow = (n) => n*n;

注意：此处也要省略return。

jQuery简介

jQuery是一个快速、简洁的JavaScript框架，是继Prototype之后又一个优秀的JavaScript代码库（或JavaScript框架）。jQuery设计的宗旨是“write Less，Do More”，即倡导写更少的代码，做更多的事情。它封装JavaScript常用的功能代码，提供一种简便的JavaScript设计模式，优化HTML文档操作、事件处理、动画设计和Ajax交互。
jQuery的核心特性可以总结为：具有独特的链式语法和短小清晰的多功能接口；具有高效灵活的css选择器，并且可对CSS选择器进行扩展；拥有便捷的插件扩展机制和丰富的插件。

DOM学习笔记|2020.3.20

什么是DOM？

  • DOM 是 Document Object Model（文档对象模型）的缩写。
  • DOM的主体是document对象。
  • 我们通过DOM来操作html的增删改查。
  • DOM不可以操作CSS，但是可以操作样式。
  • Dom操作为我们提供了一系列接口，方便我们让js和html互动。

Dom的核心是节点。

在w3c标准中，html的一切都是节点。

看一段demo：

在HTML部分，我们在section里放了个p标签和span标签，并且把节点定位到了section，那么我们现在看看section下有多少字节点：

有五个节点！

细看这五个节点，我们可以分析出其实这五个节点翻译过来就是：

1.section标签后面部分，节点属性为text。

2.p标签里面。节点属性为p（标签）

3.p标签后面。属性text。

4.span标签里面。属性标签

5.span标签后面。属性text。

但是，我们并不是每次都要用这么复杂的字节点，我们可以只返回标签节点，毕竟这个最常用。

我们可以发现，其实字节点返回的是一个类数组。

有关类数组的知识点，可以看数组笔记。

Dom选择器

获取节点，就要用选择器，选择器还挺多的，而且各不相同。

  • document.getElementById()
    使用ID来定位唯一的一个节点。

  • document.getElementsByTagName()
    利用标签名定位一个类数组。（返回的是一串标签，如果需要精准定位，需要用[]。

  • document.getElementsByClassName()
    利用class名定位。

  • querySelector()
    css定位，返回第一个内容。

  • querySelectorAll()
    css定位，返回所有内容。

值得注意的是，css定位选择的节点并不是实时更新的，尽量用document定位。

节点类型

  • 元素
  • 属性
  • 文本
  • 注释
  • document(整个文档)

数组复习笔记|2020.3.20

数组的基本概念

  • 数组是一个对象(Object)。
  • 数组是很多属性的集合。
  • 数组默认第一位是0。
  • 数组中可以包含任意类型。
  • 数组用[]表示。
  • js的数组不会有越界错误，长度可以自定义。
  • 数组里还可以套数组。

方法

数组内部有很多自带的方法，这些方法你可以用，也可以自己重做。

我们以以下数组作为例子：

indexOf()

搜查一个元素的指定位置。

slice()

slice方法会截取数组的一部分，并返回一个新的数组。

亦或者是：

push()和pop()方法

push方法可以在数组末尾继续增加一个元素，pop方法会删除数组最后一个元素。

例：

unshift和shift

往数组头部加入元素可以用unshift()，而shift()可以把数组第一个元素截掉。

Markdown简介

Markdown是一种轻量级的标记语言，通过简单的语法，可以让文字拥有一定的格式。由于简单方便，通常用于写文档、博客等。本文就是使用Markdown+vs code进行编辑的。



Markdown标题

Markdown使用#号来标记标题。和HTML语法类似，Markdown也有1～6级标题可供选择。具体实现如下：

# 我是一级标题

## 我是二级标题

### 我是三级标题

##### 我是四级标题
###### 我是五级标题
###### 我是六级标题

从五级标题开始，字就比正文要小了。
