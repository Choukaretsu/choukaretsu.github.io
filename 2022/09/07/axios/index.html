axios的方法

目前学习的主要是四种方法：

  • GET 用于请求内容
  • POST 用于发送内容
  • DELETE 用于删除内容
  • PUT 用于修改内容

axios的配置项

设定默认配置项

axios.defaults

设置默认请求类型为GET

设置默认请求地址，函数内可省略

包括很多配置项，例如params，都可以通过defaults来设置默认值。

函数内部配置

URL

请求目标链接

method

请求方法，例如GET PUT POST

headers

自定义请求头

params

在URL中的请求参数（可能通常用于get），例如浏览器顶部的地址栏

data

对象，添加到请求体中（可能通常用于post请求）

timeout

设定请求中断时间，单位毫秒

withCredentials

申请跨域时是否需要凭证，布尔值

auth

认证信息，对象
