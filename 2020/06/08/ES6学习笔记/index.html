ES6学习笔记

ES6学习笔记

前言

学到哪里写到哪里，方便回头的时候复盘学习之路

let关键字

  • 变量不能重复声明。

    let a = 1; let a =2;

    会报错，但是var关键词不会。

  • let的作用域很小，只在代码块内实现，并且不存储在window对象里。

  • 不存在变量提升。

  • 不影响作用域链。

const声明常量

  • 值不能被修改的量被称为常量。
  • 注意事项：
    • 必须要赋一个初始值，否则报错。
    • 一般大写。
    • 对数组和对象内的修改不算对常量的修改，不会报错。（因为常量的指向地址没有改变。）

模板字符串

  • es6标准支持了新的反引号，并且支持换行符

<script> let list = `<ul> <li>炼狱小镇</li> <li>核子危机</li> <li>荒漠迷城</li> </ul>`; </script>

变量拼接

let name = 'zjl';

let out = `我的名字叫${name}`；

console.log(out)

正常输出。

箭头函数

let fn = () => {

Something...

}

特性

  • 箭头函数的this始终指向声明时所在的作用域
  • 箭头函数不能作为构造函数
  • 没有arguments变量

箭头函数的省略形式

  • 当有且只有一个形参时，可以省略小括号。

let add = n =>{

return n+n;

}

此处省略了小括号。

  • 当只有一条语句时，还可以省略花括号。

    ​ let pow = (n) => n*n;

注意：此处也要省略return。
